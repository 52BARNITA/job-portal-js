# Job Portal Web Application

A simple Job Portal web app built using JavaScript.

## Features
- Search jobs by title, company, or location
- Dynamic job cards using DOM manipulation
- Apply button with state management (apply only once)
- Clean and user-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript (ES6)

## Learning Outcome
- Fetch API
- Array methods (filter, map)
- DOM manipulation
- Event handling
- UI state management
